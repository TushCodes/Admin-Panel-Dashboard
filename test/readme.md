### TESTING FOLDER

This folder contains reusable testing assets for the JavaScript dashboard foundation.

- `test_supabase_connection.js` verifies database connection helper behavior without requiring credentials.
- Run `npm test` to execute every Node.js test case in this folder.
- `dummy_consignments.js` defines `DUMMY_CONSIGNMENTS`, the canonical 100-record test dataset matching the JavaScript `Consignment` model fields.
